Comment utiliser le Dashboard
▼
Vue d'ensemble de votre activite. Le Dashboard regroupe les informations essentielles : entonnoir de conversion, taches a venir, activite recente, prospects chauds et progression de vos objectifs mensuels.
  • Cliquez sur n'importe quel element pour acceder au detail du prospect.
  • Le bouton Mode Focus lance l'enchainement automatique de vos taches.
  • Les objectifs mensuels se configurent dans Parametres.
?
Comment utiliser le Pipeline
▼
Vue Kanban de vos prospects par etape.
  • Glissez-deposez les cartes entre les colonnes pour changer leur etape.
  • Cliquez sur une carte pour editer le prospect.
  • Filtrez par profil ou priorite en haut.
  • Le point colore en haut a droite indique la priorite (rouge=chaud, orange=tiede, cyan=froid).
  • La barre de score sous l'avatar montre l'engagement (0-100).
  • Bouton Gerer le pipeline pour personnaliser etapes/sequence/profils.

Comment utiliser les Contacts
▼
Liste complete et triable de vos prospects.
  • Cliquez sur un en-tete de colonne pour trier (clic a nouveau pour inverser).
  • Filtrez par etape, profil, priorite en haut.
  • Cochez plusieurs prospects pour des actions groupees (changer etape, supprimer).
  • Le crayon edite, la croix supprime.
  • La colonne Seq. indique l'avancement de la sequence (J+x ou jours de retard).

Comment importer vos prospects
▼
Importez vos contacts depuis Excel ou CSV.
  • Glissez-deposez votre fichier .xlsx, .xls ou .csv dans la zone.
  • Mappez les colonnes de votre fichier vers les champs du CRM (auto-detection).
  • Verifiez l'apercu, choisissez l'etape de depart et lancez l'import.
  • Cochez Ignorer doublons pour eviter les contacts deja presents (verifies par email/telephone).
  • La 1ere ligne du fichier doit contenir les en-tetes (Nom, Email, etc.).

Comment gerer vos taches
▼
Toutes les actions a faire dans les 3 prochains jours.
  • Les taches sequence proviennent de votre cadence de prospection (J+0, J+5...).
  • Les rappels programmes apparaissent avec une icone horloge.
  • Cliquez sur la coche pour valider une tache.
  • Le Mode Focus vous fait enchainer les taches une par une, sans distraction.
  • Les badges colores indiquent l'urgence : rouge=retard, orange=aujourd'hui, or=a venir.
?
Comment utiliser le Calendrier
▼
Vue mensuelle de vos rappels et taches sequence.
  • Les points colores sur les jours indiquent le nombre d'evenements (rappels, sequence).
  • Cliquez sur un jour pour voir le detail des taches/rappels et les prospects concernes.
  • Naviguez entre les mois avec les fleches.
  • Le jour actuel est mis en surbrillance.
?
Comment lire les Statistiques
▼
Analyse complete de votre pipeline.
  • Entonnoir de conversion : nombre de prospects ayant atteint chaque etape (cumulatif).
  • Par etape/profil/source/priorite : repartition de votre base.
  • Montant par etape : valeur monetaire (en milliers EUR) du pipeline.
  • Sequence : nombre de prospects ayant valide chaque touchpoint.
  • Vue globale : KPIs cles synthetiques.

Comment configurer le CRM
▼
Personnalisation et gestion du compte.
  • Compte : mode cloud (sync multi-appareils) ou local.
  • Pipeline : personnaliser etapes, sequence et profils.
  • Objectifs : definir vos cibles mensuelles (nouveaux, signes, CA).
  • Templates : emails/SMS reutilisables avec variables {prenom}, {nom}, {entreprise}.
  • Raccourcis clavier : 1-8 navigation, N nouveau, / rechercher, Esc fermer.
  • Sauvegarde : exporter/importer toutes vos donnees en JSON.
