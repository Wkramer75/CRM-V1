Comment utiliser le Dashboard
▼
Vue d'ensemble de votre prospection. Pipeline, taches a venir et activite recente.
  • Cliquez sur n'importe quel element pour ouvrir la fiche prospect.
  • Le bouton Mode Focus enchaine les appels a passer sans distraction.
  • La cadence des appels suit la sequence J+0, J+1, J+2, J+3, J+4, J+5 (6 appels max).
?
Comment utiliser le Pipeline
▼
Pipeline des appels. Chaque etape represente l'appel en cours (Nouveau, Appel 1 a 6, Booke, Signe, Perdu, Inactif).
  • Glissez-deposez les cartes entre colonnes pour changer manuellement l'etape.
  • Cliquez sur une carte pour ouvrir la fiche et enregistrer un appel.
  • Le chip indique la prochaine action attendue (J+N ou retard).

Comment importer vos prospects
▼
Importez vos contacts depuis Excel ou CSV.
  • Glissez-deposez votre fichier .xlsx, .xls ou .csv dans la zone.
  • Mappez les colonnes de votre fichier vers les champs du CRM (auto-detection).
  • Verifiez l'apercu, choisissez l'etape de depart et lancez l'import.
  • Cochez Ignorer doublons pour eviter les contacts deja presents (verifies par email/telephone).
  • La 1ere ligne du fichier doit contenir les en-tetes (Nom, Email, etc.).

Comment gerer vos taches
▼
Vos appels a passer dans les 3 prochains jours.
  • La sequence suit la cadence J+0 a J+5 (6 appels max).
  • Cliquez sur la coche pour ouvrir la boite d'appel (timer + outcome).
  • Le Mode Focus enchaine les appels avec raccourcis 1/2/3/4 pour NRP/Recontacter/Booke/Non qualifie et Entree pour valider.
  • Badges : rouge=retard, orange=aujourd'hui, or=a venir.
?
Comment utiliser le Calendrier
▼
Vue mensuelle de vos rappels et taches sequence.
  • Les points colores sur les jours indiquent le nombre d'evenements (rappels, sequence).
  • Cliquez sur un jour pour voir le detail des taches/rappels et les prospects concernes.
  • Naviguez entre les mois avec les fleches.
  • Le jour actuel est mis en surbrillance.
?
Comment lire les Statistiques
▼
Analytics Setter.
  • Taux de contact : % de prospects ayant recu au moins 1 appel.
  • Taux de book : % de bookes parmi les contactes.
  • Appels moyen avant book : efficacite de la cadence.
  • Top objections : raisons les plus frequentes de Non qualifie.
  • Heatmap : vos meilleurs creneaux horaires d'appel.
  • Performance par source : quelles sources convertissent le mieux.
?
Comment configurer le CRM
▼
Personnalisation et gestion du compte.
  • Compte : mode cloud (sync multi-appareils) ou local.
  • Pipeline : personnaliser etapes, sequence et sources.
  • Templates : emails/SMS reutilisables avec variables {prenom}, {nom}.
  • Raccourcis clavier : 1-8 navigation, N nouveau, 1-4 outcome d'appel, Entree valider, Esc fermer.
  • Sauvegarde : exporter/importer toutes vos donnees en JSON.
